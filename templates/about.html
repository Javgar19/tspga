Project information

Author: Javier García Cañadillas (jgarciacanadillas@gmail.com)

Github repository: []

Frontend: HTML, CSS, Bootstrap and Javascript. For the map, the Leaflet library has been used.

Backend: Python (Flask)

Main algorithm: Genetic Algorithm. Each individual chromosome is an array of integers. Each integer represents a location and the order in the array is the visit order. The population size and mutation rate are adjusted based on the amount of locations. The crossover method used for the individuals breeding is Order Crossover. The termination criteria is met when the best individual does not change in a consecutive number of generations.

Overall description: When a location is added to the map, the waypoint coordinates are stored in an array in JS. When the button in clicked, this array is sent to the backend server where it is processed with Python. In this moment, a Genetic Algorithm class object is instanciated and initiated with the corresponding parameters. The algorithm is run and the solution is parsed to a JSON format and sent back to the frontend, where using Javascript and Leaflet the solution path is drawn.
